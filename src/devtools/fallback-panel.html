Pink Sock Security Analyzer

This is a fallback panel that appears when the main panel cannot be loaded. Please follow the troubleshooting steps below.

Troubleshooting Steps

1 Reload the extension

Go to brave://extensions/, find Pink Sock, and click the reload icon.

2 Check the console for errors

Open DevTools for this panel by right-clicking and selecting "Inspect" to check for specific errors.

3 Verify the extension is working

Click the Pink Sock icon in the toolbar to see if the popup appears correctly.

4 Try in a fresh browser profile

Sometimes browser extensions can conflict. Try loading the extension in a new browser profile.

5 Refer to documentation

Check the TROUBLESHOOTING.md file in the extension directory for more specific guidance.
